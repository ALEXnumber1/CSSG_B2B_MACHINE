# Skill: Gestión de Reuniones y Streaming B2B (Jitsi Interface)

Este skill define el protocolo para realizar reuniones de consultoría de alto nivel y transmisiones seguras dentro de la plataforma CSSG. El objetivo es proporcionar una alternativa gratuita, privada y profesional a herramientas como Zoom o Google Meet, integrada directamente en el ecosistema B2B de la empresa.

## 1. Infraestructura de Streaming (Jitsi Meet)

La plataforma utiliza la API externa de **Jitsi Meet**, lo que garantiza:
- **Gratuidad Total:** Sin límites de tiempo ni suscripciones.
- **Privacidad:** Las salas se crean de forma dinámica y no requieren cuentas de usuario.
- **Cifrado:** Todas las transmisiones están protegidas por TLS y SRTP.

## 2. Cómo Iniciar una Reunión

1.  **Navegación:** Ir a la sección `/reunion-segura` desde el menú principal.
2.  **Generación de Sala:** La plataforma genera automáticamente un nombre de sala único (ej: `CSSG-Secure-Room-1234`).
3.  **Configuración de Entrada:** Por defecto, el audio y video entran silenciados para garantizar la privacidad inicial.

## 3. Protocolo para el Consultor (Host)

Cuando un consultor de CSSG inicia una reunión, debe seguir estas pautas:
-   **Invitación:** Copiar el enlace de la sala Jitsi y enviarlo al socio/cliente vía WhatsApp o Email corporativo.
-   **Branding:** Recordar al cliente que la reunión ocurre dentro de la infraestructura de CSSG, lo que refuerza la percepción de seguridad tecnológica.
-   **Herramientas:** Utilizar la opción de "Compartir Pantalla" para mostrar los Informes PESTEL o los Resultados del Análisis de Riesgo en tiempo real.

## 4. Ventajas Estratégicas B2B

| Característica | Beneficio para CSSG |
| :--- | :--- |
| **No requiere Descargas** | El cliente entra desde el navegador, eliminando fricciones técnicas. |
| **Interfaz Limpia** | El diseño "Premium Dark" mantiene la estética de lujo de la marca. |
| **Control de Seguridad** | El consultor puede establecer contraseñas adicionales dentro de la interfaz de Jitsi. |

## 5. Escenarios de Uso Recomendados

1.  **Auditoría Táctica:** Revisión de puntos ciegos en planos o fotos de seguridad.
2.  **Gestión de Crisis:** Coordinación rápida entre la directiva del cliente y los oficiales de CSSG.
3.  **Presentación de Presupuestos:** Explicación detallada de la tecnología propuesta (drones, cámaras térmicas, etc.).

## 6. Resolución de Problemas (FAQ)

-   **"No puedo ver mi cámara":** Asegurarse de que el navegador tiene permisos para acceder a la webcam en el dominio de la plataforma.
-   **"El audio se entrecorta":** Jitsi ajusta automáticamente la calidad según la conexión. Se recomienda usar cable de red para reuniones críticas.
-   **"¿Es seguro?":** Sí, Jitsi usa WebRTC con cifrado de extremo a extremo (E2EE) si se activa en los ajustes de seguridad de la sala.
