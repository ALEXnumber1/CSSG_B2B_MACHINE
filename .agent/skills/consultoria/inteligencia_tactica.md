# Skill: Inteligencia Táctica, Gestión de Riesgos y Reportes B2B

Este skill define la metodología técnica y el estándar de calidad para la generación de informes, análisis de riesgos y consultoría táctica dentro de la plataforma CSSG.

## 1. Marcos de Referencia y Estándares
Para mantener el nivel "Luxury Tech" y la autoridad técnica, todos los análisis deben alinearse con:

*   **ISO 31000:2018:** Gestión de riesgos - Directrices.
*   **ANSI/ASIS ORM.1:2017:** Gestión de Riesgos, Resiliencia y Seguridad.
*   **Estándar COVENIN (Venezuela):** Normativa local para sistemas de protección y seguridad industrial.
*   **Metodología PESTEL:** Análisis de factores Políticos, Económicos, Sociales, Tecnológicos, Ecológicos y Legales.

## 2. Metodología de Análisis de Riesgos (Modelo CSSG)

### Matriz de Priorización (Impacto x Probabilidad)
Se utiliza una escala de 1 a 5 para ambos ejes:

| Nivel | Probabilidad | Impacto (Consecuencia) |
| :--- | :--- | :--- |
| **5: Crítico** | Evento casi seguro / Frecuente | Pérdida total de activos, fatalidades o cierre legal. |
| **4: Alto** | Probable en el corto plazo | Daño operativo severo, pérdida reputacional mayor. |
| **3: Medio** | Posible / Ocasional | Interrupción temporal, costos financieros moderados. |
| **2: Bajo** | Poco probable / Raro | Daño menor, requiere ajustes operativos mínimos. |
| **1: Mínimo** | Altamente improbable | Impacto insignificante en la operación. |

### Cálculo del Score de Riesgo:
`Score = (Probabilidad * 0.4) + (Impacto * 0.6)`
*   **4.0 - 5.0:** Mitigación Inmediata (Prioridad 1)
*   **2.5 - 3.9:** Acción Planificada (Prioridad 2)
*   **1.0 - 2.4:** Monitoreo Periódico (Prioridad 3)

## 3. Inteligencia Geo-Referenciada (Contexto Venezuela)
Categorización de zonas para informes tácticos:

*   **Zonas de Alto Valor (ZAV):** Chacao, Las Mercedes, La Castellana. Enfoque en: Vigilancia electrónica, seguridad VIP, prevención de extorsión.
*   **Zonas Industriales (ZI):** Guarenas, Valencia, Maracay. Enfoque en: Protección perimetral, control de inventarios, logística segura.
*   **Zonas de Riesgo Crítico (ZRC):** Áreas con alta incidencia delictiva. Enfoque en: Inteligencia previa, rutas seguras, escolta táctica.

## 4. Estructura de Informes PESTEL de Seguridad

Al generar un informe PESTEL para un cliente, seguir este orden lógico:

1.  **Resumen Ejecutivo:** "The Bottom Line Up Front" (BLUF). Lo que el CEO debe saber en 30 segundos.
2.  **Análisis de Factores:**
    *   **Político:** Estabilidad local, relaciones diplomáticas (clave para clientes G7).
    *   **Económico:** Inflación, costos de reposición de activos, mercado de divisas.
    *   **Social:** Conflictividad social, sindicatos, tendencias delictivas zonales.
    *   **Tecnológico:** Estado de las telecomunicaciones, vulnerabilidades de red locales.
    *   **Ecológico:** Riesgos naturales, acceso a servicios básicos (agua/luz).
    *   **Legal:** Reformas en leyes de seguridad privada o laboral.
3.  **Mapa de Recomendaciones:** Acciones tácticas inmediatas, a corto y largo plazo.
